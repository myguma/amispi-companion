# Character Asset Directory — default

このディレクトリにキャラクタースプライト画像を配置します。

## ファイル形式

| ファイル名 | 状態 | 必須 |
|-----------|------|------|
| `idle.png` | 待機中 (フォールバック) | 推奨 |
| `touched.png` | クリック反応 | 任意 |
| `sleep.png` | スリープ | 任意 |
| `waking.png` | 起床 | 任意 |
| `thinking.png` | 思考中 | 任意 |
| `speaking.png` | 発話中 | 任意 |

## 解決順序

1. `{state}.png` が存在すれば使用する
2. 存在しなければ `idle.png` を使用する
3. `idle.png` も存在しなければ SVG フォールバックを表示する

## 推奨スペック

- サイズ: 160×160px 以上 (縦長でも可)
- 背景: 透明 (PNG)
- スタイル: 自由 (アニメーションは CSS で付加する)

## 注意

このディレクトリの画像はキャラクター設定の一部です。
コードベースにキャラクター名・ロア・個性を書き込まないでください。
キャラクター差し替えはこのディレクトリの内容を変更するだけで完了します。
