# 無明が見ているもの

## 無明が気づけること (デフォルト)

- 現在時刻 (朝/昼/夕/深夜の挨拶に使う)
- しばらく入力がないこと (idle 状態の判定)
- 今アクティブなアプリの種類 (ide / browser / media / game など)
- 全画面アプリ中かどうか (全画面中は黙る)
- Desktop のファイル数 (増えすぎたら一言)
- Downloads のファイル数 (増えすぎたら一言)

**この情報は PC の外に出ません。**

---

## 許可すると見えるもの (デフォルト OFF)

- アクティブウィンドウのタイトル (設定 → 権限 でON)
- ファイル名 (将来実装予定)
- 指定フォルダの README / TODO (将来実装予定)

---

## 見ていないもの (変更不可)

- ファイルの中身
- 画像・PDF・動画の内容
- メール本文
- ブラウザ閲覧履歴
- パスワード・秘密情報
- クリップボードの内容
- 画面の常時OCR
- キーボードの入力内容

---

## 外部送信

- クラウド AI 送信: **OFF (デフォルト)**
- 送信する場合は必ず確認を取る
- 何を送るかを事前に表示する

---

## 操作権限

無明はファイルを変更・削除・移動しません。
コマンドを実行しません。
メールを送りません。
SNS に投稿しません。

---

## 観測の仕組み

無明は Windows API を通じてローカルのみで動作します。
取得した情報は PC の外に出ません。
アプリを終了するといつでも止まります。

---

## いつでも黙らせられます

- 右クリック → 「静かにする」
- Ctrl+Shift+Space で非表示
- タスクトレイ右クリック → 終了
